# Prompts — 시스템/대화 프롬프트 관리

OpenClaw 자동매매 시스템의 **시스템 프롬프트**와 **AI(Claude) 대화요약·프롬프트**를 한 폴더에서 구분해 관리합니다.

## 폴더 구조

| 파일 | 용도 |
|------|------|
| `system-prompt.md` | 시스템 프롬프트 (에이전트/봇 기본 지침, 경로·Cron·API 규칙 등) |
| `ai-prompt.md` | AI 프롬프트 (작업 지시, 체크리스트, 재사용 질문 템플릿) |
| `YYYY-MM-DD-claude-대화요약.md` | 해당 날짜 Claude 대화요약 (현황, 할 일, 명령어 등) |

## 사용 방법

- **시스템 프롬프트**: 에이전트 설정이나 “항상 이렇게 행동해라” 같은 고정 지침을 `system-prompt.md`에 적어 두고 참고/복사.
- **AI 프롬프트**: “지금 해야 할 작업”이나 “이렇게 점검해줘” 같은 요청 문구를 `ai-prompt.md`에 정리해 두고 대화 시 붙여 넣기.
- **대화요약**: 주기적으로 `YYYY-MM-DD-claude-대화요약.md`를 만들어 서버 현황, 할 일, 명령어를 기록.

## 예시 요약 파일

- `2026-02-25-claude-대화요약.md` — 2026-02-25 시스템 현황 및 할 일 정리

# AI 프롬프트 (AI Prompt)

> OpenClaw 자동매매 시스템용 AI(Claude/ChatGPT 등) 대화 프롬프트를 여기에 관리합니다.

## 용도

- 일회성 작업 지시 (예: “차트에 인터벌 버튼 추가해줘”)
- 대화 요약·체크리스트
- 재사용할 질문/지시 템플릿

## 작성 예시

```
다음 확인하고 수정해줘:
1. btc/btc_dashboard.py에서 /api/stocks/chart/{code}에 interval 파라미터 있는지 확인
2. STOCKS_HTML에 interval-btn 클래스 있는지 확인
3. 없으면 interval 버튼·API·JS 추가
4. 수정 후 대시보드 재시작 명령 실행
5. curl로 interval-btn 노출 확인
```

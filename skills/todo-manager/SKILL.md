# 할일 관리 (Todo Manager)

할일 파일: /home/node/.openclaw/workspace/brain/todos.md

## 명령어
- "할일 추가: [내용]" → todos.md에 체크박스로 추가 (- [ ] 내용)
- "할일 목록" → 미완료 할일만 보여줌
- "할일 완료: [내용]" → 해당 항목을 [x]로 변경
- "오늘 할일" → 오늘 날짜 태그된 할일 + 미완료 전체

## 파일 형식
```markdown
# 할일 목록

## 미완료
- [ ] 2026-02-15 치과 예약하기
- [ ] 2026-02-16 프로젝트 제안서 작성

## 완료
- [x] 2026-02-14 OpenClaw 설치 ✅
```

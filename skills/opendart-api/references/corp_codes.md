## 조회 스크립트

종목코드가 위 목록에 없는 경우 `scripts/get_corp_code.py`를 사용:

```bash
# 프로젝트 루트에서 실행 (.env 파일에 DART_API_KEY 필요)
source .venv/bin/activate
python3 scripts/get_corp_code.py 035420
# Output: 035420: 00266961
```

## 주의사항

- corp_code는 8자리 숫자 문자열 (0 패딩)
- 상장폐지/신규 상장 시 변경될 수 있음
- 최신 정보는 DART corpCode.xml API로 확인
